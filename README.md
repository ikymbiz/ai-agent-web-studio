# AI agent Web Studio

## 概要・説明

**AI agent Web Studio** は、ブラウザ上で完結する次世代のAI駆動型Webアプリケーション統合開発環境（IDE）です。

ユーザーはAIとチャット形式で作りたいWebアプリの要件を定義し、ワンクリックで完全なHTML/CSS/JSコードを生成できます。生成されたコードは内蔵のエディタで即座に編集・プレビューが可能で、さらにGitHubへのプッシュおよびGitHub Pagesを通じた公開（デプロイ）までをシームレスに行うことができます。

スマートフォンでの利用（スワイプでのタブ切り替えなど）にも最適化されており、Gemini、Claude、GPT、Grokといった主要な最新LLMを自由に切り替えて使用できる強力なツールです。

## 主な機能一覧

* **💬 AIチャット（要件定義）**
  * 自然言語によるWebアプリの要件定義と設計提案。
  * カメラ撮影、画像ギャラリー、テキストファイルのマルチモーダル添付に対応。
  * 「役割（システムプロンプト）」のカスタマイズによる柔軟なAIペルソナ設定。
* **📝 コードエディタ**
  * CodeMirrorを搭載したシンタックスハイライト付きエディタ。
  * ローカルファイルのアップロード/ダウンロード機能。
  * GitHubからの既存コードのインポート機能。
* **🌐 リアルタイムプレビュー**
  * 生成・編集されたコードを即座にブラウザ上でプレビュー。
  * プレビュー画面から直接AIに修正依頼を出せるデバッグパネル機能。
* **🔗 パイプライン（プロンプトチェーン）**
  * 「要件定義 → コード生成 → レビュー → 自動修正」といった一連のタスクを自動化。
  * 前のステップの出力をテンプレート変数（`{{prev_output}}`など）で次のステップに渡す連携機能。
  * 途中で人間の確認を挟む「承認ゲート（Approval Gate）」機能。
* **🚀 GitHub デプロイメント**
  * ワンクリックでGitHubの新規リポジトリ作成およびコードのプッシュ。
  * プロジェクトを解析して `README.md` を自動生成。
  * GitHub Pagesへの自動デプロイと公開URLの取得。
* **⚙️ マルチAI・バックグラウンド生成**
  * Gemini, Claude, OpenAI (GPT), xAI (Grok) の各APIに対応。
  * Service WorkerとWeb Workerを活用し、長時間のコード生成をバックグラウンドで実行（タブを閉じても生成を継続可能）。

## 使用技術・ライブラリ

本プロジェクトはビルドツールを必要とせず、単一または最小限のファイルで動作する軽量なアーキテクチャを採用しています。

* **フロントエンド**: HTML5, CSS3 (ネイティブCSS変数), JavaScript (Vanilla JS)
* **ローカルデータベース**: [Dexie.js](https://dexie.org/) (IndexedDBラッパー) - 設定や会話履歴、パイプラインの保存
* **コードエディタ**: [CodeMirror 5](https://codemirror.net/5/) - エディタUI、検索・置換機能
* **マークダウン処理**: [marked.js](https://marked.js.org/), [DOMPurify](https://github.com/cure53/DOMPurify)
* **バックグラウンド処理**: Web Workers (AIストリーミング処理), Service Workers (バックグラウンドコード生成)
* **連携API**:
  * Google Gemini API
  * Anthropic Claude API
  * OpenAI API
  * xAI Grok API
  * GitHub REST API

## セットアップ・使い方

### 1. インストール（セットアップ）
本アプリはブラウザ上で直接動作するため、複雑な環境構築は不要です。

1. プロジェクトの `index.html` をブラウザで開くか、ローカルサーバー（Live Serverなど）経由で開きます。
2. 画面下部の **「⚙️ SETTINGS（設定）」** タブを開きます。
3. **🔑 API キー設定** にて、使用したいAIプロバイダー（Gemini, Claude, GPT, Grok）のAPIキーを入力し、画面下部の「設定を保存」をクリックします。
4. （任意）GitHub連携を使用する場合は、GitHubの「Personal Access Token (権限: `repo`)」を取得し、GitHub設定欄に入力します。

### 2. 基本的な使い方
1. **要件を伝える (CHAT)**:
   「💬 CHAT」タブの入力欄に「ダークモードのTodoアプリを作りたい」等のアイデアを入力し、飛行機ボタン(✈️)を押します。
2. **コードを生成する**:
   AIが仕様を提案し、合意するとチャット上に **「⚡️ コードを作成 (BUILD)」** ボタンが表示されます。これをクリックするとコード生成が始まります。
3. **確認と修正 (PREVIEW / CODE)**:
   生成が完了すると自動的に「🌐 PREVIEW」タブに移動します。
   * 修正が必要な場合は、プレビュー画面下部の入力欄から「ボタンの色を赤にして」と修正依頼を送ります。
   * 手動でコードを微調整したい場合は「📝 CODE」タブで直接編集できます。
4. **公開する (Deploy)**:
   「📝 CODE」タブ右上の **「🚀 Deploy」** をクリックすると、GitHubリポジトリの作成からGitHub PagesでのWeb公開までをアプリ内から一括で行うことができます。
5. **高度な自動化 (Pipeline)**:
   「⚙️ SETTINGS」でパイプラインを構築し、「💬 CHAT」タブの **「▶」ボタン** を押すことで、要件定義からコードレビューまでの一連のAIワークフローを自動実行できます。
